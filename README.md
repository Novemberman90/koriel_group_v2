We developed a website for an engineering company that designs and manufactures industrial refrigeration equipment.
The website was built using the WordPress CMS.
Despite a limited budget, we implemented features for language switching, content management, and meta tags.
Technologies: HTML, SCSS, JS, Gulp, PHP, WordPress, ACF, CPT, CF7
